# PyCalc — 🧮 Kivy Calculator App

Простой, стабильный и отзывчивый калькулятор с графическим интерфейсом (GUI), написанный на Python с использованием фреймворка **Kivy**. Проект разработан с разделением архитектуры на интерфейс (`main.py`) и изолированную бизнес-логику (`logic.py`).

## 🚀 Особенности и возможности

* **Защита от дурака (Валидация ввода):** Калькулятор не позволяет вводить несколько операторов подряд (`+-`), ставить лишние запятые в одном числе или спамить нулями в начале строки.
* **Безопасный подсчет:** Вычисления проходят через встроенный парсер с полной обработкой исключений (`try/except`). При делении на ноль или синтаксической ошибке приложение не падает, а выводит `ERROR`.
* **Умная работа с числами:** * Автоматическая замена точек на запятые и наоборот для корректного отображения и вычисления.
    * Красивое отображение целых чисел (автоматически отрезает некрасивые `.0` при делении вроде `4 / 2`).
* **Дополнительные функции:** Поддержка быстрого высчитывания квадратного корня (`√`), смена знака числа (`+/-`) и посимвольное удаление (`←`).
* **Рефакторинг (DRY):** Ввод всех 10 цифр реализован через одну универсальную функцию `press_num` с динамическим определением контекста кнопки через Kivy `instance`.

---

## 🛠️ Архитектура проекта

Проект разделен на два основных модуля:
1. **`main.py`** — инициализация Kivy-приложения, настройка главного менеджера экранов (`ScreenManager`), разметка элементов через комбинацию `BoxLayout` (вертикальный каркас) и `GridLayout` (сетка кнопок 4х5), а также привязка событий (`.bind()`).
2. **`logic.py`** — чистые функции обработки сигналов от интерфейса, математические вычисления и парсинг текста.

## 📦 Инструкция по установке и запуску

Для запуска проекта на локальном компьютере должны быть установлены Python и библиотека Kivy.

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/awelare/CalculatorKivy.git](https://github.com/awelare/CalculatorKivy.git)
   cd CalculatorKivy
2. **Установите зависимости:**

pip install kivy
4. **Запустите приложение:**

python main.py
